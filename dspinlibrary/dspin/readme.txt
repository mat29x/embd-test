@par Example Description 

This example shows how to use dSPIN firmware library. Main program starts with 
clock configuration (system clock config SYSCLK to 24MHz).
The SYSCLK frequency is selected by user in clock.h file.

All used peripherals are then initialized, dSPIN chip is configured (registers are
set with requested parameters).

Different commands are then shown to perform requested operation.


@par Directory contents 

  - dspin/inc/stm32f10x_conf.h    Library Configuration file
  - dspin/src/stm32f10x_it.c      Interrupt handlers
  - dspin/inc/stm32f10x_it.h      Header for stm32f10x_it.c
  - dspin/inc/clock.h             System clock setup related header
  - dspin/src/clock.c             System clock source file
  - dspin/inc/dspin.h             dSPIN definitions header
  - dspin/src/dspin.c             dSPIN routines source file
  - dspin/inc/dspin_config.h      dSPIN configuration parameters (this file can be generated by the GUI)
  - dspin/inc/main.h              Main header file
  - dspin/src/main.c              Main program


@par Hardware and Software environment 

  
  - This example has been tested with :
      EVAL6470H-DISC board
      EVAL6472H-DISC board.

   
@par How to use it ? 

In order to make the program work, you must do the following :
- Create a project and setup all project configuration
  - For the DISCOVERY KIT : 
    - Device : ST STM32F105xB
    - Use CMSIS library
    - Preprocessor defined symbols :
      - STM32F10X_CL
      - USE_STDPERIPH_DRIVER
      - ST_DSPIN_6470H_DISCOVERY
    - Preprocessor include directories :
      - $PROJ_DIR$\..\..\..\..\..\..\stm32f10x\CMSIS\CM3\DeviceSupport\ST\STM32F10x
      - $PROJ_DIR$\..\..\..\..\..\..\stm32f10x\STM32F10x_StdPeriph_Driver\inc
      - $PROJ_DIR$\..\..\..\..\..\dspin\inc
    - Debugger :
      - ST-LINK, run to main
      - Verify download, use flash loader, override default .board file with FlashSTM32F105xB.board
- Add the required Library files :
  - startup_stm32f10x_cl.s
  - system_stm32f10x.c
  - misc.c
  - stm32f10x_exti.c
  - stm32f10x_flash.c
  - stm32f10x_gpio.c
  - stm32f10x_rcc.c
  - stm32f10x_spi.c
  - stm32f10x_tim.c
- Link all compiled files and load your image into target memory
- Run the example
